# Code4Change Women - RPL Certification Platform

** Architecture**

This platform empowers rural artisans by bridging the gap between their unverified skills and government recognition. 

### 1. Digitally assess and validate skills
**What we built:** Upload video/audio → Whisper-Tiny transcribes → AI pre-scores → validator approves.

### 2. Accessible certification
**What we built:** Instant, tamper-proof digital certificates (keccak256 hash). No paperwork.

### 3. Connect to schemes
**What we built:** Auto-maps certification to PM Vishwakarma + PMKVY RPL links and unlocks local job opportunities based on the verified certification.

### 4. Rural + low-literacy access
**What we built:** Voice-first UI. No typing, only audio/video uploads.

### 5. Multilingual documentation
**What we built:** Hindi/Kannada/English transcription so officials can verify fast.
